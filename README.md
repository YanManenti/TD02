**Required Software**
<br/>
[.NET 8](https://dotnet.microsoft.com/en-us/download)

**Launch Server** 
<br/>
*Install .NET https certificate*
<br/>
`dotnet dev-certs https --trust`
<br/>
*Navigate your cmd to the project folder, then:*
<br/>
`dotnet watch --launch-profile https`
